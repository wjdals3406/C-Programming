다항식의 덧셈
1) 차수, 계수 배열 생성 -> 인덱스로 접근
2) 연결리스트로 (차수, 계수) 하나의 노드로 구성

원형 연결 리스트와 단순 연결 리스트의 가장 큰 차이
: 원형 연결 리스트는 모든 노드에서 모든 노드로 접근할 수 있음
: 데이터 삽입할 때 O(n) -> O(1)로 시간 복잡도 감소

원형 연결 리스트 : head를 맨 앞이 아닌, 맨 뒤의 노드를 가리키게 함


<시간 복잡도 비교>
                    삽입    |     삭제
                first, last | first, last
단순연결리스트    O(1), O(N) |O(1), O(N)
원형리스트        O(1), O(1) |O(1), O(N)-> 마지막 직전 노드를 찾아가야하기 때문
